# privacy_chat
Aplicativo de chat privado desenvolvido com NodeJS integrado com banco de dados MySQL, Socket.IO, React Native e ReactJS. Nesta aplicação foi utilizada tecnologias como: express para desenvolvimento de APIs, autenticação por token JWT, nodemailer para envio de emails, cloudinary para salvamento de imagens na nuvem, redux para gerenciamento de estado da aplicação, async-storage para salvamento e recuperação de dados do usuário e entre outros.
